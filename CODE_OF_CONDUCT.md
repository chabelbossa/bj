# Code Of Conduct

DossierBJ accueille les contributions respectueuses, prudentes et orientées preuves.

## Attendu

- Rester courtois, précis et constructif.
- Critiquer les idées et les changements, pas les personnes.
- Respecter les limites du MVP : pas de données personnelles, pas de promesses administratives non sourcées.
- Signaler clairement les incertitudes, surtout sur les démarches publiques.

## Non Accepté

- Harcèlement, insultes, attaques personnelles ou intimidation.
- Publication de données personnelles, documents sensibles ou secrets.
- Ajout volontaire d'informations administratives trompeuses, inventées ou non sourcées.
- Spam, promotion hors sujet ou comportements qui rendent la contribution difficile pour les autres.

## Signalement

Ouvrez une issue privée si le dépôt le permet, ou contactez les mainteneurs par le canal indiqué dans le profil du projet. Les mainteneurs peuvent masquer, fermer ou supprimer tout contenu qui met en risque les utilisateurs ou la qualité des sources.
